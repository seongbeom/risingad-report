# 카카오모먼트 API 연동 — 재대행사 셋업 가이드

대상: **재대행사** (앱을 재대행사 명의로 소유) · 작성 2026-06-15
방식: 옵션 A — 재대행사가 앱 소유, 개발자는 팀멤버로 참여

> 진행 전 준비물: ① 재대행사 **사업자등록번호** ② 재대행사 **카카오계정**(광고주 광고계정 권한 보유한 계정) ③ 개발자 카카오계정: **__________** (← 팀멤버로 초대할 계정, 개발자가 알려줌)

---

## STEP 1. 앱 생성
1. `developers.kakao.com` 접속 → 재대행사 카카오계정으로 로그인
2. 상단 **[내 애플리케이션]** → **[애플리케이션 추가하기]**
3. 앱 이름(예: `카페24-카카오연동`), 사업자명 입력 → 저장

## STEP 2. 비즈 앱 전환 (사업자 등록)
1. 만든 앱 클릭 → 좌측 **[비즈니스]** 메뉴
2. **[비즈 앱 전환]** → 재대행사 **사업자등록번호 / 사업자 정보** 입력
3. 검수 완료까지 대기 (보통 1~2영업일)

## STEP 3. 앱 소유자 본인인증
1. 우상단 카카오계정 → **[계정 설정]** → **[본인인증]**
2. 휴대폰으로 본인인증 완료 (앱 소유 카카오계정 명의)

## STEP 4. 카카오모먼트 API 권한 신청  ⚠️ 제일 먼저, 승인 3~5영업일
1. 앱 관리 페이지 → **[앱]** → **[추가 기능 신청]**
2. **카카오모먼트 API** 사용 권한 신청
   - 담당 카카오 마케터가 있으면 마케터 통해, 없으면 카카오모먼트 공지의 신청 폼으로
3. 승인 통보 대기 (영업일 3~5일) — **이 단계가 가장 오래 걸리니 STEP 2~3과 병행해 먼저 걸어둘 것**

## STEP 5. 개발자를 앱 팀멤버로 초대 (키 안 넘겨도 됨)
1. 앱 설정 → **[팀 관리]** (또는 [앱 권한 관리])
2. 개발자 카카오계정 **__________** 을 멤버로 초대
3. → 개발자가 콘솔에서 REST API 키 등을 직접 확인 (키 별도 전달 불필요)

## STEP 6. 광고주 광고계정 권한 확인 (비즈니스 동의용)
- 비즈니스 토큰 발급 시, **동의하는 카카오계정이 그 광고주 광고계정의 마스터/멤버(또는 대행사 영업권)** 여야 함
- 재대행사 카카오계정이 연동 대상 **광고주들의 카카오모먼트 광고계정**에 권한이 있는지 확인
- 권한 없으면: 해당 광고계정에서 재대행사 계정을 멤버로 추가하거나 영업권 수락
- (대상 광고계정 목록 정리 — adAccountId는 권한만 있으면 개발자가 API로 조회)

---

## 완료 후 개발자(나)에게 넘길 것
1. **앱 팀멤버 초대 수락 완료** (STEP 5) — 그럼 키는 내가 콘솔서 확인
2. **카카오모먼트 API 권한 승인 완료** 여부 (STEP 4)
3. **연동 대상 광고주 광고계정 목록** + 재대행사 계정 권한 보유 확인 (STEP 6)
4. 비즈니스 동의(OAuth) 진행 시, 권한 가진 재대행사 카카오계정으로 1회 동의

## 개발자(나) 작업 (병행)
- 비즈니스 토큰 발급/자동 재발급 로직
- 카카오모먼트 API로 매장별 성과(노출·클릭·광고비·전환·매출) 수집
- `accounts` 테이블 컬럼 + 계정관리 입력란 + 효율시트 매핑 + 수집 잡 스케줄

---

### 핵심 요약
- 앱은 **재대행사 명의** (사업자·본인인증·API권한신청 전부 재대행사)
- 개발자는 **팀멤버로 참여** → 키 안 넘겨도 됨
- **STEP 4(API 권한 신청)를 가장 먼저** — 승인 3~5일
- 데이터 조회 권한 = 재대행사 카카오계정이 광고주 광고계정 멤버/영업권 보유 → 비즈니스 동의 1회
